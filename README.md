# work-out-routin- 🏋️‍♂️ FitFlow - 나만의 맞춤형 운동 루틴 생성기
**FitFlow**는 사용자가 입력한 요일과 운동 부위를 기반으로 최적화된 운동 루틴을 자동 분배하고, 각 부위별 추천 운동의 상세 가이드를 제공하는 **반응형 단일 페이지 애플리케이션(SPA)**입니다. 
헬스 앱 감성의 세련된 다크 테마(Sleek Dark Mode)와 글래스모피즘 효과가 적용되어 프리미엄 피트니스 앱과 같은 시각적 만족감을 선사합니다.
---
## ✨ 주요 기능 (Key Features)
1. **개인 맞춤형 루틴 설정**
   * 사용자 이름을 입력하여 개인화된 결과 표(`[이름]님의 운동 루틴`)를 출력합니다.
   * 운동 부위(가슴, 등, 어깨, 하체, 팔)와 요일(월~일 주말 포함)을 멀티 칩 형태로 쉽고 직관적으로 선택합니다.
2. **스마트 순환 매칭(Round-Robin) 알고리즘**
   * 선택된 요일 수와 운동 부위 수의 상관관계를 자동으로 분석하여 루틴을 공평하게 분배합니다.
   * **요일 수 ≥ 부위 수**: 요일을 순환하며 부위를 하나씩 배정(일부 요일은 자연스럽게 휴식일로 지정)합니다.
   * **요일 수 < 부위 수**: 선택한 요일에 모든 부위가 포함되도록 균등하게 겹쳐 분배합니다.
3. **인터랙티브 운동 가이드 (아코디언 토글)**
   * 루틴 생성 후, 표 안의 운동 부위(예: 가슴, 등)를 클릭하면 추천 운동 5가지 팝업 모달이 노출됩니다.
   * 추천 리스트에서 운동명을 누르면 부드러운 슬라이드 아웃 애니메이션과 함께 **올바른 자세와 타겟 근육, 부상 방지 팁**이 담긴 상세 운동 방법이 펼쳐집니다.
4. **로컬 스토리지(LocalStorage) 데이터 보존**
   * 루틴을 한 번 생성하면 입력값과 생성된 표 데이터가 브라우저에 자동 저장됩니다.
   * 페이지를 새로고침하거나 브라우저를 껐다 켜도 직전 루틴 정보가 자동으로 완벽하게 복원됩니다.
5. **완벽한 반응형 웹 디자인**
   * 모바일, 태블릿, 데스크톱 등 다양한 디바이스 화면에 유연하게 대응합니다. (데스크톱 2단 레이아웃 ➡️ 모바일 1단 스택 레이아웃 전환)

## 🛠️ 기술 스택 (Tech Stack)
* **Markup**: HTML5 (웹 접근성 준수, 구조화된 Semantic HTML)
* **Styling**: CSS3 (Vanilla CSS 변수 테마 시스템, Glassmorphism, CSS Transition, Responsive Flexbox & Grid)
* **Logic**: Vanilla JavaScript (ES6+ 문법, DOM 조작, 로컬 스토리지 연동, 순환 매칭 로직)
* **Libraries**: [Lucide Icons](https://lucide.dev/) (Modern & Clean SVG Icons)
* **Fonts**: Google Fonts - Inter (Body), Outfit (Headings)
